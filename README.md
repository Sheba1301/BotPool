# 🌿 NotRelease — тестовые сборки

Данная ветка содержит **экспериментальные и нестабильные версии** проекта.  
Здесь публикуются последние изменения, которые ещё не прошли полное тестирование.

---

## 🧪 Для кого эта ветка?

- 🛠 **Разработчики** — чтобы следить за ходом разработки и тестировать новые функции.
- 🧪 **Тестировщики** — готовые сообщать об ошибках и предлагать улучшения.
- 👀 **Любопытные пользователи** — желающие попробовать новейшие возможности до официального релиза.

---

## ⚠️ Важно знать

- Ветка **NotRelease** может содержать **баги, неполные функции и нестабильную работу**.
- Использование в production-среде **не рекомендуется**.
- Все изменения здесь — это черновики, которые могут кардинально меняться.
